# Flask API Project

This is a simple Flask API with a single endpoint:

- `GET /hello` returns a JSON greeting.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```

The server will start on http://127.0.0.1:5000
